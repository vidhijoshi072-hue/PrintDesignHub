PrintDesignHub

PrintDesignHub is a modern MERN-stack based marketplace platform that connects freelance printed-label designers with manufacturing companies. The platform helps designers showcase and sell their creative label designs while enabling businesses to browse, communicate, and purchase professional product labels through a centralized system.

🚀 Features
User Authentication & Authorization
Designer Portfolio Management
Upload & Manage Label Designs
Company Dashboard
Responsive UI Design
Secure JWT Authentication
CRUD Operations for Designs & Users
REST API Integration
Modern and User-Friendly Interface

⚙️ Installation
# Clone repository
git clone <repository-link>

# Move into project folder
cd PrintDesignHub

# Install dependencies
npm install

# Run frontend
npm start

# Run backend
npm run server
